‚ÑπÔ∏è How It Works

  • You start with 20 credits as a guest user.
  • Each validation attempt costs 1 credit.
  • You can change your guest name anytime.
  • Select a gateway: SK Based, Stripe Auth, or AutoShopify.
  • Enter card data in the format: card_number|mm|yy|cvv
  • For AutoShopify, optionally provide a Shopify site URL.
  • When credits run out, use the "Reset Session" button to start over.

üîó Gateway Details

SK Based

Validates cards using Stripe's SK (Secret Key) based authentication.

Stripe Auth

Validates cards using Stripe's standard authentication method.

AutoShopify

Validates cards against Shopify stores. Requires a valid Shopify site URL.
